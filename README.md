run file (game.py) in the (main_) folder

the controls are:

    A - move left
    D - move right
    SPACE - jump
    ESC - pause
    I - open inventory
    LEFT MOUSE BUTTON - melee attack
    RIGHT MOUSE BUTTON - fireball

    In the inventory:

        LEFT MOUSE BUTTON - equip
        RIGHT MOUSE BUTTON - drop
        I - close inventory
